# Frontend (Next.js)

Dieses Verzeichnis ist für die Next.js-Webapp reserviert.

Lokale Initialisierung (aus der Shell, nicht durch Jarvis):

```bash
cd frontend
npx create-next-app@latest . --typescript --eslint
```

Danach können wir hier das Dashboard (Listings, Filter, Commute-Ansicht, Document Center) implementieren.
